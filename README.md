# JEST TUTORIAL: 

I am learning the basic of jest testing. 

### SET-UP:
- Create a project folder `mkdir project_name `.
- Run the command `npm init -y`. 
- Install jest with the command: `npm i -D jest`
- Change "test" to `jest --watch`
